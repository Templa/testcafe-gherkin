# Gherkin-Testcafe implementation

## Install and run test

```sh
$ cd testcafe-gherkin
$ npm install -d
$ npm run test
```

## Concurrent test

```sh
$ npm run test -- -c 2
```

Where 2 is the number of concurrent browsers
